
## Sistema de estacionamiento

### Enunciado
Desarrollar un sistema para la gestion de un pequeño estacionamiento, registrando
la entrada y salida de vehiculos, gestion de clientes y cobro de aranceles por
tiempo de estacionamiento.

### Datos del usuario y requisitos funcionales

- [x] 5 lugares disponibles para estacionar (expandible a futuro a 10)
- [x] registro de entrada y salida de clientes
- [x] datos cliente: nombre, apellido, telefono y correo
- [x] datos vehiculo: marca, modelo, color y patente
- [x] valor por hora -> actual 2.500 por hora (precio estacionamiento)
- [x] minimo 1hr de estacionamiento, luego costo de tiempo total estacionado
- [x] un cliente puede dejar mas de un vehiculo a la vez
- [x] cada vehiculo tiene su propio tiempo de estacionamiento
- [x] no hay reserva de estacionamiento
- [x] se cobra al retirar el vehiculo
